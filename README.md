## Create Environment 
```
conda create -n HelmetDetectionSystem python=3.7 -y
```
### Activate the environment 
```
conda activate HelmetDetectionSystem
```

```
https://github.com/it3037rakesh/HelmetDetectionSystem.git
```

```
cd HelmetDetectionSystem
```
